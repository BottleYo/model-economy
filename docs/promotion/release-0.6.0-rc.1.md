# Model Economy 0.6.0-rc.1 GitHub Pre-release 草案

> 历史文案：`v0.6.0-rc.1` 标签与 GitHub Pre-release 已于 2026-07-21 发布；稳定版发布后请使用 `release-0.6.0.md`。

## English

Model Economy 0.6.0-rc.1 is the first public release candidate for directory-safe cost-aware development routing in Codex.

Highlights:

- Core mode works without user-level six-role configuration and keeps native intent, planning, testing, and evidence gates.
- Optional enhanced mode installs six locally managed role definitions with inherited or explicit model mapping.
- Partial, conflicting, or outdated enhancement artifacts fail closed as degraded state.
- The read-only `status --format text|json` command reports mode and managed-file health without reading authentication or raw session data.
- Public Pages, community templates, deterministic promotion assets, and a reproducible evaluation protocol are included.

This release candidate makes no fixed Token-savings claim and does not verify model or role identity. Please report installation failures with the installation issue template and include only redacted diagnostics.

## 简体中文

Model Economy 0.6.0-rc.1 是首个面向公开软启动的候选版，用于验证 Codex 目录式安装下的成本感知开发路由。

主要变化：

- 核心模式不依赖用户级六角色配置，并保留意图、计划、测试和证据质量门。
- 可选增强模式安装六个本地受管理角色，支持继承式或显式模型映射。
- 部分缺失、冲突或过期的增强配置会进入降级状态并失败关闭。
- 只读 `status --format text|json` 报告模式和受管理文件健康状态，不读取认证或原始 session 数据。
- 包含公开 Pages、社区模板、可重复生成的推广素材与可复算评估协议。

本候选版不承诺固定 Token 节省比例，也不验证模型或角色身份。安装失败请使用安装 Issue 模板，并只提交脱敏诊断信息。

## 升级到稳定版的门槛

- Pages、支持、隐私与条款 URL 可匿名访问。
- GitHub CI 全绿，真实 Codex marketplace 安装、重复安装、升级与卸载通过。
- 五个正向、三个负向审核用例通过。
- 没有未解决的核心安装缺陷或夸大宣传文案。
