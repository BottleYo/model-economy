# Model Economy 0.5.2 GitHub Release 草案

> 这是可复制到 GitHub Release 的双语草案，不表示 Release 已创建或发布。当前工作树已进入 `0.6.0-rc.1`，不能直接标记为 0.5.2；若坚持先发 0.5.2，必须先从变更中构造独立 0.5.2 commit/tag 并重新执行发布门。

## English

Model Economy 0.5.2 prepares the community project for a public soft launch without changing its routing behavior or local state format.

Highlights:

- Added a static bilingual GitHub Pages site with public support, privacy, and terms pages.
- Added contribution guidance, a code of conduct, roadmap, pull-request guidance, and bug/feature/installation issue templates.
- Reworked first-run documentation around the formal repository installation, recommended inherited enhancement, three risk-level examples, and explicit stop/removal controls.
- Added deterministic GitHub/X and six-card Xiaohongshu assets, a demo storyboard, English/Chinese short posts, and a Chinese technical article.
- Added a promotion-period A/B protocol that records total Token and estimated cost without inferring role-level usage or promising a fixed savings percentage.

This is a community open-source project, not an official OpenAI product. Model and role identity remain unverified. GitHub Pages and community settings must be enabled by the repository owner before publication.

## 简体中文

Model Economy 0.5.2 为社区软启动补齐公开发布层，不改变核心路由行为或本地状态格式。

主要变化：

- 新增双语静态 GitHub Pages，以及公开支持、隐私和条款页面。
- 新增贡献指南、行为准则、路线图、PR 说明与缺陷/功能/安装 Issue 模板。
- 重写首次安装与体验，明确正式仓库路径、推荐 inherited 增强、三类风险示例和停用/移除方法。
- 新增可重复生成的 GitHub/X 横图、六张小红书卡片、演示分镜、中英文短帖与中文技术长文。
- 新增推广期 A/B 协议，记录总 Token 与估算成本，不推断角色级用量，也不承诺固定节省比例。

本项目是社区开源项目，并非 OpenAI 官方产品；模型与角色身份仍未验证。发布前需要仓库所有者开启 GitHub Pages、Issues 和 Discussions，并确认公开页面可匿名访问。
