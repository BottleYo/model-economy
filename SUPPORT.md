# 支持说明

## 使用帮助

- 安装失败：使用 GitHub 的“安装问题”Issue 模板。
- 可复现缺陷：使用“缺陷报告”Issue 模板。
- 使用方式、案例和一般讨论：使用 GitHub Discussions。
- 新能力建议：使用“功能建议”Issue 模板。

报告时请提供操作系统、Python/Codex 版本、公开命令、脱敏后的错误摘要和预期行为。不要附加凭据、认证缓存、原始 session、私人项目内容或完整用户路径。

## 安全问题

可能泄露数据、越权写入或绕过归属保护的问题，请按照 [安全策略](SECURITY.zh-CN.md)私密报告，不要创建公开 Issue。

Model Economy 是社区开源项目，不代表 OpenAI 官方支持渠道。Codex 产品或账号问题请使用 OpenAI 官方支持入口。
