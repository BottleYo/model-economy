Privacy

Model Economy has no telemetry, account service, analytics, cookies, or background network process. Its local CLI manages only declared Model Economy files under CODEX_HOME and does not read Codex authentication caches or raw sessions.

The optional usage command runs a user-selected CodexBar executable and publishes only bounded, whitelisted summary fields. CodexBar remains a separate third party with its own behavior and policies.

Uninstalling with the documented purge option removes Model Economy managed local configuration. Global routing is removed separately to protect shared guidance files.

简体中文

Model Economy 不包含遥测、账号服务、统计脚本、Cookie 或后台联网进程。本地 CLI 只管理 CODEX_HOME 下已声明的 Model Economy 文件，不读取 Codex 认证缓存或原始 session。

可选的 usage 命令会运行用户选择的 CodexBar，并只公开受资源限制和字段白名单保护的摘要。CodexBar 是独立第三方。
