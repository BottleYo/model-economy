Model Economy

Use strong models for decisions, not routine work.

Model Economy classifies development work by risk, applies policy-level limits to model and subagent requests, and keeps high-risk architecture and final review explicit. It does not promise fixed token savings or verify actual model identity.

How it works

Simple

Main agent, no subagent, no strong request.

Mechanical

Fixed rules only, with a check for every result.

Standard

Main-agent execution with conditional evidence gates.

Large / high-risk

Architecture and final-review gates when enhanced roles are available.

Core first, enhancement optional

Core mode

All skills work after directory installation without local six-role files. High-risk work fails closed when isolated roles are unavailable.

Enhanced mode

The local CLI optionally installs six managed role definitions. Identity remains unverified.

Install

git clone https://github.com/BottleYo/model-economy.git
cd model-economy
codex plugin marketplace add .
codex plugin add model-economy@model-economy-public

That is the complete core-mode path. The optional enhanced-mode command and removal instructions are in the repository installation guide.
