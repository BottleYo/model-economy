# 社区行为准则

我们希望 Model Economy 社区保持友善、专业、可验证和尊重隐私。

参与者应：

- 尊重不同经验、背景和观点。
- 聚焦技术事实、复现证据与可执行建议。
- 在分享日志、配置或案例前移除敏感信息。
- 接受维护者为安全、隐私和项目边界作出的合理决定。

不可接受的行为包括骚扰、歧视、威胁、公开他人私人信息、恶意消耗维护资源，以及提交凭据、认证数据或未经授权的生产信息。

一般问题可通过 GitHub Discussions 联系维护者；安全或隐私事件必须按照 [安全策略](SECURITY.zh-CN.md)私密报告。维护者可以编辑、隐藏或删除违反本准则的内容，并限制持续违规者参与项目。
