# 路线图

路线图表达方向，不承诺固定日期。

## 0.5.2：推广准备（已并入 0.6.0）

- GitHub Pages、支持、隐私和条款页面。
- 社区入口、Issue 模板和贡献说明。
- 安装首屏、首次体验和推广素材。
- 推广期 A/B 评估协议，不提前声称节省比例。

## 0.6.0-rc.1：软启动候选版（已完成）

- 已发布 GitHub 预发布版并通过标签 CI。
- 验证 Pages、Issues、Discussions 与真实 marketplace 安装。
- 准备五个正向和三个负向审核用例；推广期继续收集安装与 A/B 数据。

## 0.6.0：当前稳定版

- 核心模式与六角色增强模式。
- 只读 `status` 文本/JSON 接口。
- 核心模式下的安全降级和高风险失败关闭。
- 官方目录 Starter Prompts 与五个正向、三个负向测试材料。

## 官方目录提交（后续）

- 从 `v0.6.0` 标签重新执行五个正向和三个负向审核用例。
- 由发布者完成身份验证、地区选择和门户政策声明。
- 正式提交审核前再次核对支持、隐私与条款 URL。

## 后续候选

- 在 Codex 提供可信身份事件后验证实际角色和模型。
- 基于真实样本发布可复算的成本观察报告。
- 改善跨设备安装，但不引入后台更新、遥测或凭据管理。
