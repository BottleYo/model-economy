# 路线图

路线图表达方向，不承诺固定日期。

## 0.5.2：推广准备

- GitHub Pages、支持、隐私和条款页面。
- 社区入口、Issue 模板和贡献说明。
- 安装首屏、首次体验和推广素材。
- 推广期 A/B 评估协议，不提前声称节省比例。

## 0.6.0-rc.1：软启动候选版

- 发布为 GitHub 预发布版，不创建正式 `0.6.0` 标签。
- 验证 Pages、Issues、Discussions 与真实 marketplace 安装。
- 运行五个正向和三个负向审核用例，收集安装与 A/B 数据。

## 0.6.0：官方目录兼容稳定版

- 核心模式与六角色增强模式。
- 只读 `status` 文本/JSON 接口。
- 核心模式下的安全降级和高风险失败关闭。
- 官方目录 Starter Prompts 与五个正向、三个负向测试材料。

## 后续候选

- 在 Codex 提供可信身份事件后验证实际角色和模型。
- 基于真实样本发布可复算的成本观察报告。
- 改善跨设备安装，但不引入后台更新、遥测或凭据管理。
